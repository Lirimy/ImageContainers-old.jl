ImageContainers.ImageContainer — Type
ImageContainer{format, S}

Stores raw image data as format.

supported format = [:jlc, :png, :svg, :jpg, :jpeg, :bmp, :gif, :mp4, :webm] :jlc represents Matrix{T<:Color}.

storeimage(file::AbstractString)

Stores data as format. Returns ImageContainer{format, typeof(data)}.

Examples
